# Сайт лекарств

Простой сайт для управления лекарствами на Laravel.

## Установка

1. Установите зависимости:
```
composer install
```

2. Создайте файл .env (скопируйте из .env.example если есть)

3. Создайте базу данных SQLite:
```
touch database/database.sqlite
```

4. Запустите миграции:
```
php artisan migrate
```

5. Запустите сервер:
```
php artisan serve
```

Откройте браузер: http://localhost:8000

## Функции

- Каталог лекарств (доступен всем)
- Регистрация и вход в систему
- Добавление нового лекарства (только для авторизованных)
- Редактирование лекарства (только для авторизованных)
- Просмотр детальной информации о лекарстве
- Удаление лекарства (только для авторизованных)

## Поля лекарства

- Название (обязательно)
- Описание
- Цена (обязательно)
- Количество (обязательно)
- Категория
- Производитель
- Срок годности
